Built with Bootstrap 4

I used this free bootstrap theme to see what it's like to set up a bootstrap site using a theme. First impression, not bad. It also uses Gulp, which is kinda cool, so I'm playing around with that a little bit too. Goofy ipad photo was also included.

Overview

Definitely a quick and easy way to get a site up. Almost too easy...but it is pretty nice to have all of the components and stuff ready to go and/or customize. I didn't change much, but the seperate sass files make it pretty easy to do so and then gulp conveniently compiles and minifies it all, which is cool. I can see why bootstrap is so popular, eventhough I'm not a huge fan of the standardized this is a website look.

Mountains

This was honestly way easier than I thought it would be so I'm hesitant to even put this site up, but I figured it could show that I am aware of the more traditional website design that isn't really shown in my other projects.

Bus

I'm also glad I was finally able to use the gulp command. Something about that naming/branding and how well it aligns to the functionality of it is really satisfying to me for some reason.
Grunt is cool too though!

Another Mountain

At the very least this project has given me a place to share my old vacation pictures that have just been sitting on my computer. Thanks for checking it out!
